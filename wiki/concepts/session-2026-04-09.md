## Key Decisions

### Undo: Native Illustrator (not custom UndoStack)
**Decision**: Removed `SetSilent(true)` and custom `UndoStack` segment restoration. Let Illustrator manage undo natively via SDK tool context bundling (each mouse drag = one undo context).

**Why**: Custom undo crashed because `SetPathSegments` from the timer callback conflicted with Illustrator's internal undo state. Codex adversarial review identified stale `AIArtHandle` pointers surviving across working-mode sessions as the root cause. Multiple fix attempts (clear on boundaries, frame validation, RedrawDocument removal) all failed. The SDK docs confirm tool mouse selectors are bundled into a single undo context — native undo is the correct approach.

### MCP Integration: Direct HTTP Bridge (not CEP relay)
**Decision**: Added 6 HTTP routes to the C++ plugin's bridge at `localhost:8787` (`/api/inspect`, `/api/create_path`, `/api/create_shape`, `/api/layers`, `/api/select`, `/api/modify`). Claude can now control Illustrator directly without CEP panels.

**Why**: The CEP → WebSocket → ExtendScript path was slow, required panel installation, and was a separate dependency. The C++ plugin already has an HTTP server. Direct SDK calls through the bridge are faster and more reliable. Condvar-based synchronous request/response mechanism for thread-safe SDK access.

### Trace: Direct vtracer Call (not MCP server)
**Decision**: TraceModule shells out to `.venv/bin/python -c "import vtracer; ..."` directly instead of HTTP POST to MCP server.

**Why**: The MCP server uses stdio protocol (not HTTP). The trace endpoint was POSTing to itself. Direct Python call via `popen()` is simpler and works without MCP server running.

### Normal-to-Perspective Pipeline
**Decision**: Added `ClusterNormalDirections` and `EstimateVPsFromNormals` to VisionEngine. Auto-match perspective now uses dual detection (Hough lines + normal direction clustering) with confidence combination.

**Why**: Normal maps encode surface orientation at every pixel. Gradient direction peaks reveal surface planes. The perpendicular of each gradient peak is the edge direction for that plane family. Parallel edges converge at a VP. This gives VP detection even when Hough fails (organic shapes, soft edges).
