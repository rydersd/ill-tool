## Motivation
Quick reference for what tools exist and where to find them when building pipelines or adding new capabilities.

## Overview

245+ tools across 8 Adobe applications. Illustrator accounts for 75% (185 tools) reflecting the project's focus on illustration automation.

## Illustrator Tool Categories (185 tools)

### Drawing & Path Manipulation (32 tools)
Shapes, smart shapes, paths, booleans, offsets, welds, contour scanning, vectorization, curve fitting, bezier optimization, stroke profiles, line weight, text, image trace, reference underlays.

### Character Rigging & Skeleton (28 tools)
Skeleton annotation/build, landmark detection (ML + axis-based), body part labeling, joint rotation/geometry, IK solving, part binding, hierarchy building, deformation/weight/interaction zones, rig controllers, constraint system.

### Pose & Animation (25 tools)
Pose capture/application/interpolation, keyframe timeline, motion paths, onion skinning, secondary motion, anticipation markers, batch pose/rig, timing curves, gesture/action lines.

### 3D & Spatial Pipeline (22 tools)
3D reconstruction (TripoSR, InstantMesh, TRELLIS.2), mesh face grouping, 3D-to-2D projection, spatial pipeline orchestrator, feedback loop, DWPose delta extraction, drawing orchestrator/session, compare/auto-correct/diffvg-correct.

### Storyboarding & Scene (30 tools)
Panel management, scene graph/composition, script-to-storyboard, camera/lighting notation, staging, transitions, beat sheets, audio sync, EDL/OTIO export, revision tracking, multi-character, continuity checking, dialogue layout.

### Color & Style (10 tools)
Color sampling/clustering/scripting, shading inference, tonal analysis, style guide/transfer, proportion checking/grids.

### Segmentation & Analysis (12 tools)
Part segmentation (ML), shape analysis, contour labeling/nesting, symmetry detection, negative space, physics hints, CV confidence, failure detection.

### Classification & Learning (8 tools)
Object classifier, part questioner, size ranker, active learning, cross-object patterns, correction learning, trace workflow, DWPose delta extractor.

### Export & Format (15 tools)
Format hub, Live2D/Spine/Rive/Lottie export, batch export, PDF, character sheets/turnarounds, AE keyframe/comp/animatic export, project dashboards.

### Template & Asset (12 tools)
Shape recipes, template matching/search/scaling/export/inheritance, asset registry/versioning/3D library, prop manager.

### Utility & Meta (8 tools)
New document, director markup, layer organization, visual debugger, smart dispatcher, environment, thumbnail grid/promote, undo checkpoint.
